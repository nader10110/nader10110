### ⚙️ How I Work

> I don't just analyze data — I follow a structured process that turns noise into insight.

```
 RAW DATA  ──▶  CLEAN  ──▶  EXPLORE  ──▶  ANALYZE  ──▶  VISUALIZE  ──▶  DECIDE
```

**1 · Collect** — CSVs, databases, APIs, spreadsheets  
**2 · Clean** — handle nulls, duplicates, and outliers with Python & SQL  
**3 · Explore** — EDA to find distributions, correlations, and anomalies  
**4 · Analyze** — CTEs, Window Functions, and statistical summaries  
**5 · Visualize** — Power BI dashboards & Python charts  
**6 · Deliver** — clear insights that drive real decisions ✅
